# Sistema de Cadastro de Usuários

##  O que é
Sistema em Python que permite cadastrar, listar e buscar usuários usando JSON.

##  Funcionalidades
- Cadastro de usuários
- Busca por nome
- Armazenamento em JSON

##  O que aprendi
- Manipulação de JSON
- Estrutura de dados
- Organização de código

##  Como rodar
python main.py
